Brief:
    Implementation of Pacman game using C++ and Qt ui framework.

Authors: 
    Šimon Benčík <xbenci01>
    Roman Poliačik <xpolia05>

What was implemented:
- Loading maps from file - if not loaded a default one is used
- Saving games to replays
- Base movement of Pacman with keyboard
- Movement of ghosts using A* algorithm
- Logging of the game (replays) using both modes

Extension (Interactive elements):
- Collectible - fruits that pacman eats
- Score
- Key count

What wasn't implemented:
- Movement of pacman using mouse click
